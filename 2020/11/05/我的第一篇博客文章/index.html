##第一次使用Hexo框架搭建属于自己的博客

一些Hexo代码: hexo c #清理一下
hexo g #生成
hexo d #部署到远程Github仓库

炅哥建议每次generate之前先清理一次。总结下来，每次自己更新完博客后的操作顺序应该是首先c，其次是g部署，最后再是d到github。总的来说，也算是一次新鲜的尝试，希望自己以后能够坚持写博客。
